# HTML(HyperText Markup Language)
index.html - 진입점

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
  </head>
  <body>
    <span>안녕하세요. 반갑습니다.</span>
    <span>안녕하세요. 반갑습니다.</span> // 한 줄로 나옴
    <div>김지은입니다.</div>
    <div>김지은입니다.</div> // 엔터 엔터 계속 내려감
  </body>
</html>
```
