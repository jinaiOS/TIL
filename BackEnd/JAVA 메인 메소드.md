# JAVA 메인 메소드
```java
public class Main {
    // public static void main 메소드 실행
    public static void main(String[] args) {
        System.out.println("Hello world!");
    }
    
    public static void myMethod() {
        System.out.println("Hello world!");
    }
}
```

- **main** method가 실행 시킨다!
- myMethod 작동 안 함
